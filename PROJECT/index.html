About the Agency

This dataset is provided by the New York City Police Department (NYPD) through NYC Open Data. NYC Open Data makes government information available to the public to encourage transparency, research, and community engagement.

Dataset Description

The NYPD Arrest Data dataset contains records of arrests made throughout New York City. Information includes arrest date, borough, age group, race, gender, offense category, and location.

Importance of the Dataset

Arrest data helps people understand crime and law enforcement activity across NYC. It supports research, promotes transparency, and helps identify trends among different communities and boroughs.

Information Page

The Information page explains the dataset, its variables, and provides details about the records being analyzed.

Analysis Page

The Analysis page contains charts and visualizations that allow users to explore arrest trends. Users can filter data and compare different groups and boroughs.

Map Page

The Map page displays arrest locations across New York City, helping users visualize where arrests occur geographically.

Project Summary

This website makes NYPD arrest data easier to understand by combining information, charts, filters, and maps into one interactive platform. Users can explore trends, compare boroughs, and gain insights from public safety data.
